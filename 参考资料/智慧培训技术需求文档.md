# 智慧培训技术需求文档

#### 模块一：申安学堂

##### 一、培训课件分级管理

### 1、分级目的

实现安全培训内容的**精准化、场景化、高效化**，避免“一刀切”式培训，确保不同层级人员掌握与其职责匹配的安全知识与技能，提升培训实效性与合规性。

### 2、四级培训课件体系设计

| 培训对象 | 核心目标 | 内容重点 |
| --- | --- | --- |
| 管理层 | 强化安全领导力与责任意识 | \- 安全生产法律法规（新安法、刑法修正案）<br>\- 企业安全主体责任<br>\- 安全投入与资源配置<br>\- 事故追责与舆情应对 |
| 省区安全/行政/运营综合岗 | 掌握区域安全统筹与落地能力 | \- 转运中心安全检查标准<br>\- 应急预案编制与演练组织<br>\- 承包商安全管理 |
| 转运中心专职/兼职安全员 | 提升现场风险识别与干预能力 | \- 机械伤害、高处坠落、火灾等典型风险辨识<br>\- 安全巡查技巧与隐患闭环流程<br>\- 安全培训组织方法<br>\- 智慧安全系统操作 |
| 分拣员、装卸工、司机等 | 掌握岗位安全操作与应急处置 | \- 岗位SOP（如叉车、月台、充电区）<br>\- PPE正确佩戴<br>\- “三不伤害”原则<br>\- 初起火灾扑救、紧急疏散 |

### 3、课件形式与交付要求

课程形式包含**Word、Pdf、ppt、视频**

所有课件需标注：**版本号、更新日期、适用范围、编制人**。

### 4、系统权限与管理流程

#### 4.1.上传权限

**仅限**：生产安全部指定管理员（如安全培训主管）

可操作：上传、编辑、下架

#### 4.2下载与使用权限

**管理层**：可查看并下载课件，不可修改；

**省区人员**：可查看并下载适用课件，不可修改；

**中心安全员**：可下载课件用于一线培训；

**一线员工**：仅通过扫码/链接观看视频或PDF，无下载权限。

#### 4.3.下载记录追踪

系统自动记录每次下载行为，包含：

*   下载人姓名、工号、所属省区/中心
    
*   下载时间、课件名称、版本号
    
*   备注用途（如“用于3月班组培训”）
    

## 模块二：三级安全教育培训线上化管理

### 一、模块目标

实现公司级、转运中心级、班组级三级安全教育的**全流程数字化管理**，支持：

*   **线上扫码签到**（主流方式，实时高效）
    
*   **线下纸质补录**（兼容历史或特殊场景）
    

确保所有培训记录可追溯、可验证、可归档，满足安全生产法规及内部审计要求。

### 二、核心功能设计

#### 1、培训内容固化管理

*   由**生产安全部管理员**在后台维护三级标准培训内容模板： 
    
    *   公司级安全教育（如安全制度、法律法规）
        
    
    *   转运中心级教育（如场地风险、设备操作）
        
    *   班组级教育（如岗位SOP、应急处置）
        

#### （2）线上扫码签到（主流程）

*   **二维码生成**：
    
    *   中心安全员创建一场培训（选择级别+内容+时间+地点+人数），系统自动生成**唯一动态二维码**，有效期默认2小时。
        
*   **员工操作**：
    
    *   扫码后自动识别身份（通过钉钉集成HR系统），页面显示： 
        
    *   员工信息：姓名、岗位、部门、入职时间（自动带出，不可编辑）
        
    *   勾选项：“本人已接受本次培训” √ 是 / × 否
        
    *   若选“是”，弹出电子签名面板，手写签名确认
        

#### （3）纸质记录补录（辅助流程）

*   **适用场景**： 
    
    *   网络异常无法扫码
        
    *   外包人员无系统账号
        
    *   历史纸质档案数字化
        
*   **操作权限**：
    
*   仅限**中心安全员**操作。
    
*   **补录流程**： 
    
    管理员进入“纸质记录补录”页面；
    
    手动填写或批量导入（Excel模板）以下字段： 
    
    *   姓名、工号、岗位、部门、入职时间
        
    *   培训级别、培训内容（下拉选择固化模板）
        
    *   培训日期、是否完成（是/否）
        
    
    **上传纸质签名页扫描件或照片**（JPG/PDF，≤10MB）；
    
    提交后系统标记为“线下补录”，并记录**补录人+补录时间**。
    
*   **校验规则**： 
    
    *   工号必须存在于HR系统（防止虚构人员）；
        
    *   同一员工同一级别培训，30天内不可重复补录。
        

#### （4）培训完成状态判定

*   系统自动追踪每位员工的三级培训状态： 
    
    *   公司级：已完成（线上/线下）
        
    *   转运中心级：未完成
        
    *   班组级：未开始
        
*   **仅当三级均为“已完成”**，才视为“三级教育闭环”。
    

#### （5）数据导出与归档

*   管理员可按**省区/****中心/时间段**筛选记录；
    
*   支持一键导出为 **Word 文档**（用于存档）或 **Excel**（用于分析）；
    
*   Word 格式包含： 
    
    *   员工基本信息
        
    *   三级培训内容摘要
        
    *   电子签名图 或 纸质签名扫描件缩略图
        
    *   完成时间、录入方式标识
        

### 三、权限控制

| **角色** | **权限** |
| --- | --- |
| 生产安全部管理员 | 创建/编辑培训内容、查看全量数据、导出报告 |
| 省区管理员 | 查看本省区数据、导出报表，**不可补录或修改** |
| 中心安全员 | 扫码组织培训、补录本中心人员纸质记录、查看本中心数据 |
| 普通员工 | 仅扫码签到，无后台访问权限 |

### 四、非功能性要求

*   **安全性**：签名数据加密存储，操作日志保留≥1年；
    
*   **兼容性**：支持、钉钉、申通内部APP扫码；
    
*   **性能**：单场培训支持500人同时扫码签到，响应<2秒；
    
*   **合规性**：符合《生产经营单位安全培训规定》及集团EHS档案标准。
    

### 五、后续扩展建议（V2.0）

*   对接HR系统自动同步新员工名单，触发三级培训任务；
    
*   增加培训视频播放强制观看+答题环节；
    
*   生成个人安全培训电子证书。
    

## 模块三：全员安全考试

### 一、功能目标

*   实现**全岗位、全覆盖、全周期**的安全知识考核；
    
*   通过**自动化组卷、一键下发、智能提醒**，提升考试效率；
    
*   确保员工掌握岗位安全应知应会内容，强化安全意识；
    
*   为安全绩效评估、风险预警提供数据支撑。
    

### 二、适用对象（全员覆盖）

| **人员类型** | **考试重点** |
| --- | --- |
| 管理层（总裁、省总、中心经理） | 安全法规、主体责任、应急管理<br>岗位SOP、PPE使用、应急处置、基础安全禁令、现场风险认知<br>岗位SOP、PPE使用、应急处置、基础安全禁令、现场风险认知 |
| 总部及省区其他人员 |
| 安全员（综合负责人、省区安全员、中心安全员）及总部其他人员 |
| 一线员工（分拣、装卸、司机、仓管）及外包/临时人员 |

✅ 支持按**岗位、部门、中心、省区**灵活分组下发考试。

### 三、题库管理

#### 1. 题型支持

单选题、多选题、判断题

#### 2. 权限控制

*   **管理员**（生产安全部）：可新增、编辑、审核、停用题目；
    
*   **省区/中心**：可申请新增本地化题目，需总部审核后生效。
    

#### 3. 题目属性

每道题需包含：

*   题干、选项、正确答案
    
*   解析（用于考后学习）
    
*   关联法规或制度条款（如《安全生产法》第XX条）
    

### 四、考试流程（全流程线上化）

1.  **创建考试**
    
    *   管理员：填写考试名称、时间、对象、题型组合、及格线（如90分）；
        
    *   可设置“随机抽题”（如从100题中抽20题）或“固定试卷”。
        
2.  **一键下发**
    
    *   点击“发布”，系统自动推送考试通知至目标人员钉钉；
        
    *   员工在规定时间内登录系统完成答题。
        
3.  **在线答题**
    
    *   支持手机/PC端；
        
    *   防作弊机制：禁止切屏（可选）、限时作答、随机题序。
        
4.  **自动判分 & 结果反馈**
    
    *   提交后即时显示得分并参考合格线（90分）显示是否通过考试、错题及解析；
        
    *   通过者和未通过者可查看题目，错题集，未通过者支持补考（补考次数3次）
        
5.  **电子归档**
    
    *   自动生成个人成绩单 + 全员汇总报表；
        
    *   支持导出Excel/Word，用于安全档案留存。
        

### 五、智能提醒机制（确保参与率）

| **时间节点** | **提醒方式** | **系统动作** |
| --- | --- | --- |
| 考试开当天 | 钉钉推送通知 | 推送提醒：“您有一场安全考试，请于XX前完成” |
| 考试截止前24小时 |  | 对未参考人员二次提醒 |
| 截止后仍未完成 |  | 自动标记“缺考”，通知其直属主管，并计入安全考核 |
| 补考安排 |  | 可设置自动开启补考通道（如24小时内） |

### 七、系统集成建议

*   **身份对接**：与HR系统/统一身份认证平台打通，自动同步人员岗位信息；
    
*   **数据看板**：在智慧安全系统首页展示“本月考试完成率”“TOP3薄弱知识点”；
